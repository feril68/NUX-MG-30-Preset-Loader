# nux-mg30-preset-loader

Electron desktop app (Vue 3 + TypeScript) for loading and managing NUX MG-30 presets.

## Requirements

- Node.js LTS
- npm
- A MIDI-capable environment (and MG-30 connected) for device workflows

## Setup

```bash
npm install
```

## Run in Development

```bash
npm run dev
```

## Scripts

- `npm run dev`: Start Electron + Vite in development mode
- `npm run start`: Run preview mode
- `npm run lint`: Run ESLint
- `npm run format`: Format files with Prettier
- `npm run typecheck`: Run Node + Vue type checks
- `npm run build`: Typecheck and build app bundles
- `npm run build:unpack`: Build unpacked distributable
- `npm run build:win`: Build Windows installer/package

## MIDI Debug Logging (Optional)

Use only when troubleshooting MIDI detection/connection:

- `DEBUG_MIDI=1`: Enables main-process MIDI debug logs
- `VITE_DEBUG_MIDI=1`: Enables renderer-side MIDI debug logs

PowerShell:

```powershell
$env:DEBUG_MIDI="1"
$env:VITE_DEBUG_MIDI="1"
npm run dev
```

Git Bash:

```bash
DEBUG_MIDI=1 VITE_DEBUG_MIDI=1 npm run dev
```

## Packaging Notes

- Product name: `NUX MG-30 Preset Loader`
- Windows target uses NSIS installer
- Linux targets: `AppImage`, `snap`, `deb`
- Build artifacts are generated by `electron-builder` in its output directory (commonly `dist/` unless overridden)
